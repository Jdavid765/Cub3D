## Description
Cub3D is a graphics project inspired by the world-famous Wolfenstein 3D game. The goal of this project is to create a dynamic, first-person 3D representation of a maze using ray-casting principles. 

Built in C using the miniLibX graphics library, this project serves as a deep dive into graphical mathematics, window management, event handling, and rendering 3D perspectives from a 2D map.
